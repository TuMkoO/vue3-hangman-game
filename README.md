# Hangman

Учебный проект игры Hangman.

**Использованные технологии:**  
:small_orange_diamond: Vue 3  
:small_orange_diamond: Typescript

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
